Tentang Saya

Saya Devi Aryani, mahasiswa Politeknik Negeri Cilacap dari program studi D3-Teknik Informatika. Saya adalah seorang Web Developer yang antusias dan memiliki keterampilan desain yang kuat. Dengan pengalaman dalam mengembangkan antarmuka pengguna yang responsif dan intuitif, saya mampu mengubah konsep desain menjadi aplikasi web yang fungsional dan menarik. Keahlian saya mencakup penggunaan teknologi seperti HTML, CSS, JavaScript, serta berbagai framework dan pustaka frontend.


Selain keahlian teknis, saya juga memiliki latar belakang yang solid dalam desain grafis. Saya terbiasa menggunakan alat desain seperti Adobe Creative Suite (Photoshop, Illustrator, CorelDraw, InDesign), Canva, Figma, dan Sketch untuk prototipe dan desain UI/UX yang efektif, poster, desain feeds sosial media, pamflet, dan berbagai materi visual lainnya. Kombinasi keterampilan ini memungkinkan saya untuk tidak hanya membuat aplikasi web yang fungsional, tetapi juga menarik secara visual


Dengan perpaduan antara keterampilan teknis dan estetika, saya berkomitmen untuk menciptakan pengalaman pengguna yang optimal dan menyenangkan. Saya selalu mengikuti perkembangan tren dan teknologi terbaru dalam industri untuk memastikan bahwa pekerjaan saya selalu up-to-date dan relevan. Saya siap untuk berkontribusi dalam tim yang dinamis dan inovatif, serta bersemangat untuk terus belajar dan berkembang di bidang Web development dan desain.
